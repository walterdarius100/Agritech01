# Checklist courte — Agri-tech V1

Avant une mise en ligne ou une PR touchant le site live :

- [ ] Vérifier l’accueil sur desktop et mobile.
- [ ] Vérifier `services.html`, `formations.html`, `actualites.html` et `article.html`.
- [ ] Vérifier que les articles publiés se chargent depuis Supabase.
- [ ] Vérifier que `admin.html` reste accessible uniquement aux utilisateurs autorisés.
- [ ] Vérifier que le formulaire `leadForm` envoie bien via EmailJS.
- [ ] Vérifier que les messages arrivent à `contact@agritech509ht.com`.
- [ ] Vérifier que les liens vers le formulaire utilisent `index.html#leadForm` si un lien direct vers le formulaire est nécessaire.
- [ ] Vérifier la console navigateur.
- [ ] Ne pas modifier Supabase, EmailJS, Clarity, DNS ou Cloudflare sans décision explicite.
